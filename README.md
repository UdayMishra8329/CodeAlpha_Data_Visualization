# 📊 Student Performance Data Visualization

## 📌 Project Overview
A comprehensive data visualization project analyzing 10,000 student 
academic records to uncover key factors affecting student performance 
and placement outcomes.

## 📁 Dataset
- **Source**: Kaggle — Student Performance Prediction Dataset
- **Records**: 10,000 students
- **Columns**: study_hours, attendance, sleep_hours, internet_usage,
  assignments_completed, previous_score, exam_score, placement_status

## 🔍 Key Findings
- ✅ **83.6%** of students got placed successfully
- 📚 **Study hours** is the strongest positive factor (r = 0.563)
- 🌐 **Internet usage** negatively impacts performance (r = -0.152)
- 😴 Average sleep hours: **6.5 hours/night**
- 📝 Average exam score: **86.70 / 100**
- 📊 Median exam score: **92.12** (majority score very high)

## 📊 Visualizations Created
1. **Main Dashboard** — 9 charts in one comprehensive view
2. **Distributions Analysis** — All variable frequency distributions
3. **Performance Categories** — Low / Medium / High breakdown
4. **Placement Analysis** — Placed vs Not Placed deep dive
5. **Correlation Analysis** — Factor impact on exam score
6. **Outliers Check** — Data quality verification

## 🛠️ Tools & Libraries Used
- **Python 3.14**
- **Pandas** — Data loading and manipulation
- **NumPy** — Numerical computations
- **Matplotlib** — Chart creation
- **Seaborn** — Statistical visualizations

## 🚀 How to Run

# Install required libraries
pip install pandas matplotlib seaborn numpy

# Run main analysis
python complete_analysis.py

# Run detailed visualizations
python detailed_visualizations.py
```

## 💡 Insights & Conclusions
1. Students who study more consistently score higher
2. Regular attendance has a moderate positive impact
3. Excessive internet usage correlates with lower scores
4. Previous performance is a good predictor of future scores
5. High performers have significantly better placement rates
